---
title: "Assets"
description: "The assets verification step validates an individual's financial assets and property ownership."
---

## Configuration

```json
{
    "id": "6797fd5e683fb1e8abe7d852",
    "name": "ASSETS",
    "control_flow": {
        "condition": {
            "condition_operator": "==",
            "condition_field": "assets_verified",
            "condition_value": true
        },
        "if_goto": "6798005e683fb1e8abe7d854",
        "else_goto": "6797fd5e683fb54e8abe7d852"
    },
    "provider": ["Plaid", "Flinks"]
}
```

## Config Fields

### Required Fields
- `id`: Unique identifier for the verification step
- `name`: Step type identifier (ASSETS)

### Control Flow
Determines the next step based on verification results:
- If `assets_verified` is `true`, proceeds to step `6798005e683fb1e8abe7d854`
- Otherwise, goes to step `6797fd5e683fb54e8abe7d852`

### Providers
- Primary: Plaid for asset verification and account connections
- Fallback: Flinks as secondary provider if Plaid connection fails

## Output Fields

### Status Information
- `status`: Current status of the verification (e.g., "COMPLETED")
- `assets_verified`: Whether all assets were successfully verified

### Account Information
- `accounts`: List of verified financial accounts
  - `account_id`: Unique identifier for the account
  - `institution`: Financial institution details
  - `account_type`: Type of account (e.g., "INVESTMENT", "SAVINGS")
  - `balance`: Current account balances and last update time
  - `investment_details`: Additional details for investment accounts

### Property Information
- `properties`: List of verified properties
  - `property_id`: Unique identifier for the property
  - `address`: Property location details
  - `ownership`: Type and percentage of ownership
  - `property_type`: Type of property
  - `valuation`: Property value information
  - `liens`: List of liens against the property

### Asset Summary
- `summary`: Overview of total assets and available funds
  - `total_assets`: Total value of all assets
  - `liquid_assets`: Total value of liquid assets
  - `property_value`: Total value of properties
  - `property_equity`: Total equity in properties
  - `down_payment_available`: Funds available for down payment

### Verification Details
- `verification`: Information about the verification process
  - `provider`: Which provider performed the verification
  - `verified_at`: When verification was completed
