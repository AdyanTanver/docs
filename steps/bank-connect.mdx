---
title: "Bank Connect"
description: "The bank connect step verifies additional sources of income through bank account connections and documentation, such as child support or other recurring payments."
---

## Configuration

```json
{
    "id": "6797fe5e683fb1e8abe7d853",
    "name": "BANK_CONNECT",
    "options": {
        "verify_account_balance": true,
        "analyze_transactions": true,
        "verify_account_ownership": true,
        "enable_instant_verification": true
    },
    "control_flow": {
        "condition": {
            "condition_operator": ">",
            "condition_field": "account_balance",
            "condition_value": 500
        },
        "if_goto": "6798015e683fb1e8abe7d855",
        "else_goto": "6797fd5e683fb54e8abe7d852"
    },
    "provider": {
        "name": "Plaid",
        "fallback_provider": "Flinks"
    }
}
```

## Config Fields

- `id`: Unique identifier for the verification step
- `name`: Step type identifier (BANK_CONNECT)
- `options`: Configuration options for bank account verification
  - `verify_account_balance`: Whether to verify current account balance
  - `analyze_transactions`: Whether to analyze transaction history
  - `verify_account_ownership`: Whether to verify account ownership
  - `enable_instant_verification`: Whether to enable instant account verification
- `control_flow`: Logic for the next step based on account balance
  - `condition`: Defines the decision criteria
    - `condition_operator`: Comparison operator (>)
    - `condition_field`: Field to evaluate (account_balance)
    - `condition_value`: Minimum required balance (500)
  - `if_goto`: Next step if balance requirement is met
  - `else_goto`: Next step if balance requirement is not met
- `provider`: The bank connection service provider configuration
  - `name`: Primary provider name
  - `fallback_provider`: Secondary provider if primary fails

## Output Fields

- `status`: The completion status of the verification
- `result`: The verification results containing:
  - `account_balance`: Current total balance (used for control flow)
  - `verification_id`: Unique identifier for this bank verification
  - `bank_accounts`: List of connected bank accounts
    - Account details and balances
    - Ownership verification status
    - Transaction history including:
      - Recurring deposits identification
      - Large deposit tracking
  - `additional_income`: List of verified additional income sources
    - Income type and amount
    - Frequency and verification status
    - Supporting documentation
    - Payment history and consistency
  - `verification_details`: Details about the verification process
    - Instant verification status
    - Ownership verification status
    - Provider used
    - Verification timestamp
  - `summary`: Summary of bank account activity
    - Total monthly deposits
    - Verified additional income
    - Average daily balance
- `timestamp`: Time when the verification was completed
