---
title: 'Employment Verification Step'
description: 'The employment verification step validates an individuals current employment status and income information through authorized employment verification providers.'
---

## Configuration

```json
{
    "id": "6797df82a4c4910bc0f1d490",
    "name": "EMPLOYMENT",
    "options": {
        "verify_income": true,
        "minimum_employment_duration": "6_months",
        "require_current_employment": true,
        "include_employer_contact": true
    },
    "control_flow": {
        "condition": {
            "condition_operator": "==",
            "condition_field": "employment_status",
            "condition_value": "verified"
        },
        "if_goto": "6797dfb4a4c4910bc0f1d491",
        "else_goto": "request_additional_info"
    },
    "provider": {
        "name": "Equifax"
    }
}
```

## Config Fields

* `id`: Unique identifier for the verification step

* `name`: Step type identifier (EMPLOYMENT)

* `options`: Configuration options for the verification process

  * `verify_income`: Whether to include income verification

  * `minimum_employment_duration`: Required minimum duration of employment

  * `require_current_employment`: Whether to require current active employment

  * `include_employer_contact`: Whether to include employer contact information

* `control_flow`: Logic for the next step based on verification result

* `provider`: The employment verification service provider configuration

## Output Fields

* `status`: The completion status of the verification

* `result`: The verification results containing:

  * `employment_status`: Whether the employment was successfully verified

  * `verification_id`: Unique identifier for this verification attempt

  * `employer`: Information about the employer

    * `name`: Company name

    * `address`: Company address details

    * `contact`: HR or verification contact information

  * `employment_details`: Details about the employment

    * `start_date`: Employment start date

    * `employment_type`: Type of employment (full-time, part-time, etc.)

    * `job_title`: Current position

    * `employment_duration`: Calculated duration of employment

    * `active`: Whether currently employed

  * `income`: Income information if requested

    * `salary`: Base salary information

    * `last_pay_date`: Date of most recent payment

    * `pay_frequency`: Frequency of payments

* `timestamp`: Time when the verification was completed
