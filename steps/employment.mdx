---
title: "Employment"
description: "The employment verification step validates an individual's current employment status and income information through payroll providers."
---

## Configuration

```json
{
    "id": "6798005e683fb1e8abe7d854",
    "name": "EMPLOYMENT",
    "options": {
        "require_pay_slips": true,
        "employer_confirmation": true
    },
    "control_flow": {
        "condition": {
            "condition_operator": "==",
            "condition_field": "employment_status",
            "condition_value": "verified"
        },
        "if_goto": "6798015e683fb1e8abe7d855",
        "else_goto": "6797fd5e683fb54e8abe7d852"
    },
    "provider": ["Atomic", "Argyle"]
}
```

## Config Fields

### Required Fields
- `id`: Unique identifier for the verification step
- `name`: Step type identifier (EMPLOYMENT)

### Options
- `require_pay_slips`: Whether to require recent pay slip documentation
- `employer_confirmation`: Whether to verify employment directly with the employer

### Control Flow
Determines the next step based on verification results:
- If `employment_status` is `verified`, proceeds to step `6798015e683fb1e8abe7d855`
- Otherwise, goes to step `6797fd5e683fb54e8abe7d852`

### Providers
- Primary: Atomic for employment and income verification
- Fallback: Argyle as secondary provider if Atomic verification fails

## Output Fields

### Status Information
- `status`: Current status of the verification (e.g., "COMPLETED")
- `employment_status`: Whether employment was successfully verified

### Employer Information
- `employer`: Details about the employer
  - `name`: Company name
  - `employer_id`: Unique employer identifier
  - `address`: Employer's address information

### Employment Details
- `employment_details`: Current employment information
  - `start_date`: Employment start date
  - `employment_type`: Type of employment (e.g., "FULL_TIME")
  - `job_title`: Current position
  - `status`: Current employment status

### Income Information
- `income`: Salary and payment details
  - `annual_income`: Yearly salary
  - `pay_cycle`: Payment frequency
  - `last_paycheck_amount`: Most recent payment amount
  - `last_paycheck_date`: Date of most recent payment
  - `next_paycheck_date`: Expected date of next payment

### Verification Details
- `verification`: Information about the verification process
  - `method`: How employment was verified
  - `provider`: Which provider performed the verification
  - `pay_slips_verified`: Whether pay slips were verified
  - `employer_confirmed`: Whether employer confirmed employment
  - `verified_at`: When verification was completed
