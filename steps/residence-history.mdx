---
title: "Residence History"
description: "The residence history step verifies an individual's residential history through document verification."
---

## Configuration

```json
{
    "id": "6797dfb4a4c4910bc0f1d491",
    "name": "RESIDENCE_HISTORY",
    "control_flow": {
        "condition": {
            "condition_operator": "==",
            "condition_field": "residence_verified",
            "condition_value": true
        },
        "if_goto": "6797dfe7a4c4910bc0f1d492",
        "else_goto": "6797fd5e683fb54e8abe7d852"
    },
    "provider": []
}
```

## Config Fields

### Required Fields
- `id`: Unique identifier for the verification step
- `name`: Step type identifier (RESIDENCE_HISTORY)

### Control Flow
Determines the next step based on verification results:
- If `residence_verified` is `true`, proceeds to step `6797dfe7a4c4910bc0f1d492`
- Otherwise, goes to step `6797fd5e683fb54e8abe7d852`

### Providers
No external providers - residence history verification is handled internally by Cove.

## Output Fields

### Status Information
- `status`: Current status of the verification (e.g., "COMPLETED")
- `residence_verified`: Whether the residence history was successfully verified

### Address Information
- `addresses`: List of verified addresses
  - `type`: Whether it's a current or previous residence
  - `address`: Address details including street, city, province/state, postal code
  - `duration`: Period of residence including start date, end date, and total months
  - `ownership`: Type of residence (e.g., "RENT") and payment information

### Verification Details
- `verification`: Information about the verification process
  - For each address:
    - `verified`: Whether the address was verified
    - `verified_at`: When verification was completed
    - `documents`: List of verified documents
  - Overall verification:
    - `total_months_verified`: Total months of verified residence history
    - `verified_at`: When the entire history was verified
